## 💡 **说明**

### 时间策略

| 选项 | 说明 |
|------|------|
| **平衡模式** | 在保证字幕时间和自然语速之间取得平衡 |
| **严格对齐** | 严格按照字幕的开始/结束时间配音 |
| **自然语速** | 保持自然说话速度，不受字幕时间限制 |
| **不限制** | 完全不限制配音时长 |

### 音频模式

| 选项 | 说明 |
|------|------|
| **替换原音** | 完全替换掉原视频的声音 |
| **混合原音** | 配音和原音同时播放（适合双语） |
| **压低原音** | 配音时自动降低原音音量（背景音保留） |
