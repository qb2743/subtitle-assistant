# 配音面板 UI 设计方案

**需求分析**：
1. 在主页添加"配音"面板（与转录、字幕、合成并列）
2. 支持配置：provider 选择、音色、停顿控制、API Key 等
3. 文稿匹配语言问题：Whisper 支持 99 种语言，DTW 算法语言无关
4. 单词拆分问题：需要智能分词（英文按单词边界）

---

## 📋 方案对比概览

| 方案 | 风格 | 优点 | 缺点 | 推荐度 |
|------|------|------|------|--------|
| **方案 A** | 上下分栏 | 布局清晰，参数齐全 | 占用空间大 | ⭐⭐⭐⭐⭐ |
| **方案 B** | 左右分栏 | 紧凑，适合宽屏 | 窄屏体验差 | ⭐⭐⭐⭐ |
| **方案 C** | 向导式 | 渐进引导，易上手 | 步骤多，高级用户繁琐 | ⭐⭐⭐ |

## 📊 关键问题解答

### Q1: 文稿匹配只能中英文吗？

**答案**：❌ 不是！Whisper 支持 99 种语言。

**Whisper 支持的语言**（部分）：
- 中文、英语、日语、韩语
- 法语、德语、西班牙语、意大利语
- 俄语、阿拉伯语、葡萄牙语
- 泰语、越南语、土耳其语
- ...等 99 种语言

**DTW 算法**：语言无关，只要文本能正确分词即可。

**建议改进**：
```python

  • ...（完整列表）
```

### Q2: 最大字符数会拆分单词吗？

**答案**：✅ 当前实现会有此问题！

**问题示例**：
```
原文: "This is a beautiful day"
最大 15 字符 → "This is a beaut" + "iful day" ❌ (单词被拆分)
```

**建议改进**：
```python
def split_by_word_boundary(text: str, max_chars: int, language: str):
    """智能分行：英文按单词边界，中文按字符"""
    if language == 'en':
        # 英文：在空格处断行
        words = text.split()
        lines = []
        current_line = []
        current_length = 0
        
        for word in words:
            if current_length + len(word) + 1 <= max_chars:
                current_line.append(word)
                current_length += len(word) + 1
            else:
                lines.append(' '.join(current_line))
                current_line = [word]
                current_length = len(word)
        
        if current_line:
            lines.append(' '.join(current_line))
        return lines
    else:
        # 中文/其他：按字符断行（当前逻辑）
        return [text[i:i+max_chars] for i in range(0, len(text), max_chars)]
```

**界面改进建议**：
```
每行最大字符数: [32▲▼]
☑ 英文按单词边界分行（避免拆分单词）
```

## 🎯 推荐方案

**综合推荐：方案 A（上下分栏式）**

### 理由：
1. ✅ 所有配音参数清晰可见
2. ✅ 支持展开/折叠高级选项
3. ✅ 适合各种屏幕尺寸
4. ✅ 与现有"转录"、"字幕"面板风格一致

### 实现优先级：
1. **核心功能**（必做）
   - Provider 选择下拉框
   - 音色输入框
   - API Key 输入框
   - 基础参数（timing, audio_mode）

2. **高级功能**（推荐）
   - 固定停顿开关
   - 输出选项（音轨 vs 视频）
   - 实时预览音色

3. **锦上添花**（可选）
   - 音色预览播放
   - Provider 参数说明
   - 批量配音快捷入口
