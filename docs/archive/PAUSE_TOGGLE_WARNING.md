**完成日期**：2026-06-21  
**状态**：✅ 已完成

---

## ✅ **功能说明**

当用户勾选**"固定停顿"**开关时，会弹出一个警告提示：

```
⚠️ 提示
开启固定停顿后，时间策略将失效
```

---

## 📝 **实现细节**

### 添加的代码

```python
def _on_pause_toggled(self, checked):
    """固定停顿开关切换"""
    if checked:
        InfoBar.warning(
            title="提示",
            content="开启固定停顿后，时间策略将失效",
            duration=3000,
            position=InfoBarPosition.TOP,
            parent=self,
        )
```

### 连接信号

```python
self.pause_switch = SwitchButton(self)
self.pause_switch.toggled.connect(self._on_pause_toggled)  # ← 新增
```

## 🎯 **用户体验**

### 操作流程

1. 用户在配音参数中看到"固定停顿"开关
2. 点击开关打开
3. 立即弹出黄色警告提示框（3秒后自动消失）
4. 提示内容："开启固定停顿后，时间策略将失效"

### 为什么需要这个提示？

- **避免混淆**：用户可能同时设置了"时间策略"和"固定停顿"，但不知道固定停顿会覆盖时间策略
- **明确优先级**：告知用户固定停顿的优先级更高
- **减少疑问**：避免用户疑惑"为什么时间策略不起作用"

---

## 📊 **参数关系**

### 时间策略（当固定停顿关闭时生效）

- **平衡模式**：平衡速度和时间
- **严格对齐**：严格按字幕时间
- **自然语速**：保持自然说话速度
- **不限制**：不限制时长

### 固定停顿（开启后优先级更高）

- **开启**：每行字幕之间固定停顿 N 毫秒
- **关闭**：使用时间策略
