### 3️⃣ **调整文稿匹配默认字符数**

**问题**：
- 默认值 32 字符可能偏小

**解决方案**：
```python
# 调整前
self.max_chars_spin.setValue(32)

# 调整后
self.max_chars_spin.setValue(40)  # 更合理的默认值
```

**理由**：
- 40 字符更适合大多数场景
- 中文：约 40 个汉字，符合字幕长度习惯
- 英文：约 6-8 个单词，阅读舒适
