## 🎨 **设计原则**

### 1. 字体颜色适配
- **深色背景** → 使用**白色**或浅色字体
- **浅色背景** → 使用深色字体
- 确保对比度足够，文字清晰可读

### 2. 拖拽框文字长度
- **原则**：每行不超过 8-10 个中文字符
- **方法**：长文本拆分成多行
- **效果**：避免文字被容器截断
